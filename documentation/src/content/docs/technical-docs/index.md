Welcome to the Technical Documentation for Mithril Forge. This area is intended for developers, architects, and technically interested users.

The documentation is structured according to the [Diátaxis standard](https://diataxis.fr/) and is divided into four sections:

- **[Tutorials](/documentation/technical-docs/tutorials):** Learning-oriented lessons that take you through a project step by step.
- **[How-To Guides](/documentation/technical-docs/how-to):** Goal-oriented guides that show you how to solve specific problems.
- **[Reference](/documentation/technical-docs/reference):** Information-oriented reference materials (e.g., API documentation, CLI commands).
- **[Explanation](/documentation/technical-docs/explanation):** Understanding-oriented articles about concepts, architecture, and background knowledge.
