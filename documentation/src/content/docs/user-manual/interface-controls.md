## Fast number editing

You can adjust numeric fields like HP or AC using standard text entry, the keyboard, or the mouse.

Press the `Up` or `Down` arrow keys inside the input to change the value by 1. You can also click the plus or minus buttons. Hold `Shift` while using the arrow keys or buttons to change the value by 10.

Hold `Alt` to turn any number field into a slider. The cursor changes to a vertical arrow. Click and drag up or down to scrub through values without typing. Hold `Shift` while dragging to jump in increments of 10.
