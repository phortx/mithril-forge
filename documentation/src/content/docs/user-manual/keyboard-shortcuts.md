description: A quick reference cheat-sheet for keyboard shortcuts and interface controls in Mithril Forge.

Mithril Forge is designed to be fast and keyboard-friendly for Dungeon Masters during combat. This cheat-sheet lists all available keyboard shortcuts and advanced interface controls to help you run encounters smoothly.

## Global Actions

| Action        | Shortcut / Control | Description                                                                                                           |
| :------------ | :----------------- | :-------------------------------------------------------------------------------------------------------------------- |
| **Next Turn** | `Spacebar`         | Advances the active turn down the initiative order. Can be used anywhere on the DM screen during an active encounter. |

## Number Editing (HP, AC, Initiative)

Any numeric input field in the DM view (such as current HP, Armor Class, or Initiative) supports advanced mouse and keyboard controls for rapid adjustments without typing.

| Action                        | Shortcut / Control            | Description                                                                                                         |
| :---------------------------- | :---------------------------- | :------------------------------------------------------------------------------------------------------------------ |
| **Increment/Decrement by 1**  | `Up` / `Down` Arrow           | Inside an input, press to change the value by 1.                                                                    |
| **Increment/Decrement by 10** | `Shift` + `Up` / `Down` Arrow | Inside an input, press to change the value by 10.                                                                   |
| **Slider Scrubbing**          | Hold `Alt` + Click & Drag     | Turns the number field into a slider (cursor changes to a vertical arrow). Drag up or down to scrub through values. |
| **Slider Scrolling**          | Hold `Alt` + Mouse Wheel      | Turns the number field into a slider. Scroll the mouse wheel to adjust values.                                      |
| **Fast Scrubbing/Scrolling**  | Hold `Alt` + `Shift`          | Changes values in increments of 10 while scrubbing or scrolling.                                                    |
