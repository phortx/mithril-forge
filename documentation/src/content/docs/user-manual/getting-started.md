Mithril Forge is hosted online and runs entirely in your browser. You don't need to install any software, create an account, or pay a subscription.

**Launch the app:** Go to [Mithril Forge](https://mithril-forge.up.railway.app).

Because the system uses your browser's local storage to sync data, everything is fast and private. The encounter state updates instantly across tabs without waiting on a server.

### Setting up the two screens

You run the game by opening two browser windows on the same computer:

1. Open a tab on your main laptop screen and set it to **DM view**. This is where you will control the fight.
2. Open a second tab, drag it to your external monitor or cast it to your TV, and set it to **Player view**.

For details on how the two views differ and how to manage the encounter, read the [Encounter Tracking](./guides/encounter-tracking/) guide.
